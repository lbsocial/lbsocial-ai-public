# Security and Privacy

This is a public repository. Anything posted in an issue may be visible to other people.

Please do not include:

- Passwords, API keys, access tokens, or secret values.
- Personal information such as email addresses, phone numbers, addresses, or account IDs.
- Private account details, billing details, or course-access information.
- Screenshots that show private conversations, credentials, profile data, or browser tokens.
- Internal logs, deployment settings, Cloud Run details, Firebase identifiers, or private service URLs.

If you need to report a sensitive issue, describe the problem at a high level and say that details should be shared privately. Do not paste sensitive content into a public issue.
